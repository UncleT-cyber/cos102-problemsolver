# COS 102: Problem Solving Platform

This project is a full-stack application developed for the COS 102 Problem Solving laboratory assignment. It combines C programs with a Python Streamlit web interface to solve two programming tasks:

* Right-Angled Triangle Checker
* Palindrome Number Checker

The application allows users to enter values through a simple web interface while the calculations are performed by C programs running in the background.

## 🏗️ System Architecture

The project consists of three main components:

### 1. C Backend

The core program logic is written in C. It performs the calculations for checking whether a triangle is right-angled and whether a number is a palindrome.

### 2. Input Validation

The application validates user input before sending it to the C program. This helps prevent invalid values and ensures reliable program execution.

### 3. Streamlit Frontend

The web interface is built with Streamlit. It provides a simple and user-friendly way to interact with the C programs without using the command line.
